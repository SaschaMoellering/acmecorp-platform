# AcmeCorp Platform – Project Context

AcmeCorp Platform is a reference architecture used to teach modern Java performance engineering.

The repository supports:

- a technical video course
- JVM benchmarking experiments
- cloud-native architecture demonstrations

The platform contains:

- Spring Boot services
- Quarkus services
- Docker Compose environments
- Kubernetes deployments
- JVM startup benchmarks

Primary learning topics:

- JVM startup performance
- containerized Java workloads
- cloud-native architecture
- benchmarking techniques

The repository prioritizes:

- reproducibility
- clear architecture
- production-grade patterns
