# AcmeCorp Platform

Demo platform for deep-dive videos about Java, containers, Kubernetes, and EKS Auto Mode.

Services:
- Orders Service (Spring Boot)
- Billing Service (Spring Boot)
- Notification Service (Spring Boot)
- Analytics Service (Spring Boot)
- Catalog Service (Quarkus, optional)

Includes:
- Local Docker Compose stack
- Kubernetes manifests (Kustomize-ready base)
- Helm chart for all services
